滑动窗口

发表于 2019-07-07 | 分类于 数据结构与算法 |
字数统计: 883 | 阅读时长 ≈ 4

实际上滑动窗口就是利用双指针来限制一个区间范围。因此这种方法适用于解决满足一定条件的连续区间的问题，注意这个限制条件： 连续区间。 因此适用于求连续子数组，字串等，而不适用于非连续的子序列等等。

下面通过leetcode的几个例题来详细的看下滑动窗口该怎么做

209. 长度最小的子数组

这道题题干很清晰，求一个和大于s的子数组，那么只需要控制左右指针，找到这个最小的区间即可。

知乎上有张图清晰的说明了滑动窗口解决这个问题的思路

3. 无重复字符的最长子串

由于C语言没有hash表，我们可以用数组模拟一个hashset。

实际上这道题我们通过i, j来充当滑动窗口的左右指针。对于我们的hashset：

如果hashset[s[j]] == 0则意味着这个字符没有重复的，窗口右侧向右滑动。

如果hashset[s[j]] == 1则意味着这个字符已经在窗口中存在了，我们要首先计算一下滑动这次之前的窗口大小，然后窗口左侧向右滑动， 直到窗口里面没有重复的字符为止。

1004. 最大连续1的个数 III
